Label

ResidencyAttribute

Description

The attribute of living in a particular City or Province. Unlike Ethnicity, which is permanent, Residency can change.

Superclasses (1)

  • NTNames:HumanAttribute

    • Usage

    Instances of NTNames:ResidencyAttribute can have the following properties:

    PROPERTYDESCRIPTIONRANGE
    From class owl:Thing
    NTNames:LNID Louw-Nida identifier, keying specific Greek terms (and their English translations) to their taxonomy of semantic domains. A string like "33.X'.439" identifies the term sumballo, which can be translated as "to express differences of opinion in a forceful way". In their scheme, this has domain index 33 (Communication), with sub-domain index X' (Dispute, Debate). The final integer index uniquely defines this term (but the same term may have multiple senses). The sub-domain index can be determined from the term index, but it seems helpful to include both. Note the possible values of the sub-domain index are A-Z, then A'-Z', etc.
    NTNames:nativePlace Birthplace, or place in which a person grows up. Unlike residentOf, only one possible value. residentPlace is to be preferred unless the birthplace relationship is clearly identified. suo:Region
    NTNames:politicalBelief Holding to a PoliticalBelief NTNames:PoliticalBeliefSystem
    NTNames:religiousBelief Having a ReligiousBelief as an attribute NTNames:ReligiousBeliefSystem
    NTNames:residentPlace The location a person or group of people resides in. Range values may change over time: no attempt to capture this. suo:Region
